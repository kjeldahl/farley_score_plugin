# Changelog

## Unreleased

### Added

- Ruby support: signal detection for RSpec and Minitest (RSpec mocks, Minitest::Mock, Mocha), including trivial/framework tautologies and Mock Anti-Patterns AP1-AP4
- Discovery now recognizes `spec/` directories and `*_spec.*` / `*_test.rb` naming conventions

## 1.0.0 (2026-02-27)

### Initial Release

- `/msec:farley-score` -- Autonomous test quality review with Farley Index scoring
- `/msec:farley-score-coach` -- Interactive test quality coaching with Socratic questioning
- Welcome menus for first-time users on both commands
- Pre-packaged demo report from bundled sample tests
- Signal detection for 5 languages: Java, Python, JavaScript/TypeScript, Go, C#
- Tautology Theatre analysis (mock tautologies, mock-only tests, trivial tautologies, framework tests)
- Mock Anti-Pattern detection (AP1-AP4)
- CLI calculator for deterministic scoring math
- Bundled sample project with deliberately mixed-quality tests for coaching exercises
